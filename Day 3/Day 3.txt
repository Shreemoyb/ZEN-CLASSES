Window- 
The window is the actual global object. The window object is simply like the root of them all to which document and screen are properties of. So, the window is the parent object while they are children objects.

Screen- 
The screen is just the user display area, the screen is the screen, it contains properties about the user's display. 

Document- 
The document is where the DOM is, document is the DOM containing all html elements, their classes and attributes.
The Document Object Model (DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.
